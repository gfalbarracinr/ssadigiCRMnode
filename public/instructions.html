Manual Técnico

Este es el manual técnico para la importación de Compañias, Contactos y Deals. Para tener el excel con los usuarios actualizados, descargar la plantilla en la pagina principal. A continuación se lista los pasos a seguir para hacer una importación.

Paso a paso

  1. Descargar la plantilla de la pagina principal

  2. Una vez haya descargado la plantilla, llenar los campos correspondientes en la hoja sheet 1 de la siguiente manera:

    • NOMBRE EMPRESA En este campo se indica el nombre de la empresa, este debe ser único, ya que si esta repetido no se incluira en el CRM, tampoco lo hará el contacto ni el deal

    • PAGINA WEB En este campo se indica la pagina web de la empresa

    • INDUSTRIA En este campo se indica la industria a la que pertenece la empresa, Hubspot recomienda unos tipos de industria que se encuentran en la hoja industria

    • PAIS En este campo se indica el prefijo del país donde se establece la empresa, es importante que este campo tenga una longitud menor o igual a tres para facilidad en la lectura

    • PRIMER NOMBRE En este campo se indica el primer nombre del contacto

    • SEGUNDO NOMBRE En este campo se indica el apellido del contacto

    • TELEFÓNO DE CONTACTO En este campo se indica el telefono del contacto, puede o no agregarse el indicativo del pais

    • EMAIL CORPORATIVO En este campo se indica el email corporativo del contacto, este debe ser unico para cada contacto, ya que si esta repetido, el contacto no se incluira en el CRM y el deal se hará solamente con la empresa

    • ETAPA Esta es la etapa que estará el deal que se va a adicionar, siempre debe estar en PREQUALIFIED. tenga presente errores en tipeo porque puede generar errores en la importación

    • VALOR DEAL En este campo se indica el valor del deal, este debe ser de tipo númerico

    • DEAL OWNER En este campo se indica el primer nombre del usuario que va a quedar responsable del Deal

    • DEAL OWNER ID Este campo se llenará automaticamente y corresponde al id del usuario, por eso es importante no tener errores en el punto anterior, si hay algun problema con el llenado de este dato remitase a la hoja Usuarios del excel

    • PRODUCTO En este campo se indica el producto del deal. Este campo solo puede ser llenado con los siguientes valores: CG, CM o GEN

    • NOTA En este campo se puede agregar una nota al deal

  3. Una vez completado el Excel, por favor dirigirse a la pagina principal, seleccione el archivo a subir (solo se permiten archivos excel) y envielo
  4. Cuando se envia el archivo, el ssitema le responderá con los errores encontrados en el archivo y las estadisticas del mismo (Compañias creadas, Contactos creados y deals creados)
